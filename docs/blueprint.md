# **App Name**: PropVibez

## Core Features:

- Property Listing: Allow users to list properties for sale or rent with detailed information and high-quality images.
- Property Search: Enable users to search for properties based on location (Noida, UP, India), price range, property type, and other criteria.
- AI-Powered Property Recommendations: Use AI to provide personalized property recommendations based on user preferences and search history, acting as a helpful tool.
- Interactive Map Integration: Display properties on an interactive map for easy location viewing.
- Contact Agent: Allow users to directly contact the property agent or owner through the platform.
- User Account Management: Enable users to create and manage their accounts, save favorite properties, and set up property alerts.

## Style Guidelines:

- Primary color: Gold (#FFD700) inspired by the logo, symbolizing luxury and value.
- Secondary color: Dark Green (#006400) from the logo, representing growth and stability.
- Background color: Off-white (#FAF9F6) for a clean and modern backdrop, ensuring readability.
- Body and headline font: 'Montserrat', a modern sans-serif font for a clean and professional look.
- Use geometric icons with gold and green accents, mirroring the logo's style.
- Clean and spacious layout with a focus on high-quality property images.
- Subtle animations and transitions using gold and green accents to enhance user experience.
